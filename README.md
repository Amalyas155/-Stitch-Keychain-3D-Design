# Stitch Keychain – 3D Design

## Overview

This project presents a 3D design of a Stitch-inspired keychain created in Onshape. The model was built using sketches and extrusion features to create a layered design suitable for 3D printing. Different body details were modeled as separate features to improve the overall appearance and make color separation easier during printing.

## Features

- Designed using Onshape
- Layered 3D keychain design
- Multi-feature modeling using Sketch and Extrude
- Individual details modeled separately
- Ready for 3D printing
- Colored visualization for presentation

---

## Design Process

The model was created by starting with the main body sketch, followed by multiple extrusion operations to build the overall thickness. Additional sketches were used to create the facial features, ears, eye details, body markings, and keychain loop. Each component was designed as an individual feature to improve accuracy and simplify color separation.
